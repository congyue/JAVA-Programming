CS9053 Introduction to Java

Homework 7

Focus

  • client-server application
  • GUI interface

Task

Create a "chat" program. The purpose is to allow a user on one machine to use a GUI to chat with a user on [potentially] another machine.

Details

  • Each user will have a graphical interface that allows them to type in text and that will display the ongoing dialog.
  • Host, port and user name information can have hard-coded defaults, but should be modifiable from the command line (and, if you like, through the graphical interface)
  • One user will act as the "server" in that it will sit waiting on a specified port number for the "client" to connect.
  • Once connected there is no distinction between the two. Each will send their user's name to the other and subsequently allow messages to be sent and received asynchronously.
  • This is not an excerise in multi-threading. Do not create any additional threads. Yes, just creating a gui automatically causes the gui thread to run, but you should not be creating any others. You'll have plenty of opportunity for that in the next assignment.

CS9053 Introduction to Java

Homework 5

Focus:

Given a set of classes, implement equality, hashcode, compareTo and clone.

Details:

The classes:

  • Car
  • Wheel
  • Engine

The task:

  • Implement the classes.
  • Write good test code for all features.
  • Provide reasonable fields.
  • For Car, override equality and hashcode. Implement the Comparable and Cloneable interfaces.
  • Of course, every classe should have appropriate toString method.

Specifics:

  • A Car has an engine and some number of wheels. Not all cars are built to hold four wheels, some have only three while others have more. But whatever they are built for, that is the max number they can hold.
  • When a car is built (i.e. constructed), an engine is created for it and so are the wheels that it will use.
    • The car's constructor will create the engine and all of the wheels. How? Based on the information provided. E.g., the number of wheels, their brand, etc.
  • Wheels can be removed and new ones added, as can engines.
    • When removing or adding a wheel, you must specify which wheel "position".
  • Engines have, at the least, a type, such as V6, and a horsepower.
  • Wheels have a brand and a radius.
